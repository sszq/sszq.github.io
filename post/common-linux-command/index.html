Common Linux Command

December 06, 2020
Linux

Linux commands category by the first letter of command.

A

# 命令
alias	[ei li e si]	别名
apropos	[ai po pou]		恰当的
aspell	放弃
*awk	[ao ke]	笨拙的

# 单词
aptitus	[ai pu tei tui de]	才能[等同于apt]
emerge	[i mo jay]	浮现[等同于apt]
pacman	[pa ke man]	巴曼（越南）[等同于apt]
yum		[ya mu]	美味的、极好的[等同于apt]

B

# 命令
basename
bash	GUN Boume-Again SHell
bc		echo "obase=2;sqrt(1024)" | bc  echo "scale=2;8/3" | bc
bg		ctrl+z to stop; from stop to start in bg; from stop to start to fg;
builtin	[bei ou tin]	内建用户
bzip2

# 单词
trip	[si tui pu]	脱衣、除去
suffix	[se fei ke si]	后缀
arbitrary	[a bei te rei]	任意的
precision	[pe sei shen]	精确

C

cal		display a calendar
cat
cfdisk	partition table manipulator for linux
chgrp	change group ownership
chmod	change access permissions
chown	change file owner and group
chroot	run a command with a different root directory
chkconfig	system services(runlevel)
cksum	print CRC checksum and byte counts
cmp		compare two files
comm	compare two sorted files line by line
command	run a command - ignoring shell functions
cron	daemon to execute scheduled commands
crontab	shedule a command to run at a later time
csplit	split a file into context-determined pieces
cut		divide a file into several parts

# 单词
concatenate	[kao en, kai tei nei te]	连结
partition	[pa tei xin]	划分
manipulator	[me nei piu lei te]	操纵器
ownership	[ou ne xi pu]	所有权
permission	允许，许可
divide		[dei vai de]	分

D

date
dc		desk calculator
dd		convert and copy a file, write disk headers, boot records
declare	declare variables and give them attributes
df		display free disk space
diff	difplay the differences between two files
diff3	show differences among three files
dig		DNS lookup
dir		briefly list directory contents
dircolors	colour setup fo 'ls'
dirname	convert a full pathname to just a path
dirs	display list of remembered directories
dmesg	print kernel & driver messages
du		Estimate file space usage


# 单词
briefly	[bu rei fu li]	简单的，简短的
estimate [ai si tei mei te]	估计
