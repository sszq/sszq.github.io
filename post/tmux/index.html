Tmux

December 14, 2020
Tools

tmux is a terminal multiplexer, and it can improve your work efficiency.

For me, it has less two usefully features in my common work. Firstly, I can use it work in multi process; Secondly, I can store my work directory, even if I exit the terminal, next time I can also restore the session by tmux a -t <session-name>.

memory tips

  1. All the first, know the structure information of tmux.

  2. Secondly, know every structure level has key actions.

    Tmux has three structure level

    • session
    • window
    • pane

  3. Thirdly, know the configuration.

  4. Other command

key word

prefix command key

prefix command key

Ctrl + b

help command

ctrl + b, ?

session

  1. add session

    tmuxtm new -s <session-name>
tmux # by this, you can add a default session.

  2. left with session

    tmux detach
ctrl + b, d	# same to above

  3. list session

    tmux ls	# it is common with [tmux list-session]
ctrl + b, s	# same to above

  4. into session

    After left with session, you want to into the session again by the bottom command.

    tmux attach -t <session-name/sequence>
tmux a -t <session-name/sequence>

  5. close without session

    tmux kill-session -t <session-name/sequence>
ctrl + d	# same to above

  6. switch session

    tmux switch -t <session-name/sequence>

  7. rename session

    tmux rename-session -t <old-session-name/sequence> <new-session-name>
ctrl + b, $	# same to above

window

  1. create window

    tmux new-window -n <window-name>
ctrl + b, c	# same to above

  2. switch window

    tmux select-window -t <window-name>
ctrl + b, w	# list windows, then you can select window by j, k, next enter.
ctrl + b, n	# switch to next window
ctrl + b, p	# switch to previous window
ctrl + b, <number>	# switch to <number> window

  3. close window

    tmux kill-window -t <window-name>
ctrl + b, &	# same to above

  4. rename window

    ctrl + b, ,

pane

  1. split window

    tmux spslit-window	# splict ont pane to two panes(up, down)
ctrl + b, "	# same to above
tmux split-window -h	# splict ont pane to two panes(left, right)
ctrl + b, %	# same to above
# split current pane to a alone window
ctrl + b, !

  2. move cursor amounts multiple panes

    tmux select-pane -U	# up
ctrl + b, up arrow	# same to above
tmux select-pane -D	# down
ctrl + b, down arrow	# same to above
tmux select-pane -L	# left
ctrl + b, left arrow	# same to above
tmux select-pane -R	# right
ctrl + b, right arrow	# same to above

  3. go to previous/next pane

    # go to previous pane; I had test it work in more panes.
ctrl + b, ;
# to to next pane; I had test it work in 2 panes.
ctrl + b, o

  4. switch pane location

    # current pane move to up
tmux swap-pane -U
ctrl + b, {	# same to above
# current pane move to down
tmux swap-pane -D
ctrl + b , }	# same to above

  5. close current pane

    ctrl + b, x

  6. maximum/restore pane

    ctrl + b, z
# aujust pane size
ctrl + b, ctrl + [arrow up/down/lefg/right]

  7. show time in pane

    when pressed enter, reduce.

    ctrl + b, t

  8. show pane number

    ctrl + b, q

configuration

  1. touch a file named .tmux.conf in ~ path.

  2. set heightlight in terminal.

    set -g default-terminal "screen-256color"

    `

  3. reset tmux prefix command key

    set -g prefix C-x	# reset Ctrl + b to Ctrl + x
unbind C-b
bind C-x sned-prefix

  4. set swap pane key

    bind-key k select-pane -U
bind-key j select-pane -D
bind-key h select-pane -L
bind-key l select-pane -R

  5. reload the configuration file ~/.tmux.conf

    tmux source-file ~/.tmux.conf

other command

  1. list all tmux shortcut key, and the command it relative

    tmux list-keys

  2. list all tmux commands and its parameters

    tmux list-commands

  3. list all tmux session info now

    tmux info

  4. reload current tmuxw configuration

    tmux source-file ~/.tmux.conf
