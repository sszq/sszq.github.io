Purchase receives

In purchase receives application, user can new, receive, return material. In bussiness, user can do these actions only on the purchase approve status is APPR, so if the status is not APPR, when user do that, throuth system notice user can’t do that.

Firstly, the recievts line table binding a table class, and the receive, return action is in this method, now I can create a new class and override the two method.

package hxqh.webclient.beans.receipts;

Delete Item accounts

Usually, users always tole we to delete some material in item accounts application, every item relationships many tables, so do it in datase visuble tools is too tight. Finally, I plan to write a method, then I can do it in applications by click mouse.

Firstly, I override this method by delete(), in Item.class is no this method, and it’s just have delete(long access), so program find delete() in Item.class ‘s father class. In Mbo.class have delete() method, so I wrote methos always not work.

I also override canDelete() method, becase when I delete, I don’t need too many limits.

/**
  * 删除物资台账，及相关表数据
  */
@Override
public void delete(long access) throws RemoteException, MXException {
    String itemNum = getString("itemNum");

    UdAtgUtil.print("Item delte start: " + itemNum);

    MboSetRemote itemOrgInfoSet = server.getMboSet("ITEMORGINFO", server.getSystemUserInfo());
    itemOrgInfoSet.setWhere("itemNum='" + itemNum + "'");
    itemOrgInfoSet.reset();
    System.out.println("itemOrgInfoSet.count()" + itemOrgInfoSet.count());
    if (null != itemOrgInfoSet) {
        itemOrgInfoSet.deleteAll(11L);
        itemOrgInfoSet.save();
    }
    itemOrgInfoSet.close();

    MboSetRemote itemOrgStatusoSet = server.getMboSet("ITEMORGSTATUS", server.getSystemUserInfo());
    itemOrgStatusoSet.setWhere("itemNum='" + itemNum + "'");
    itemOrgStatusoSet.reset();
    System.out.println("itemOrgStatusoSet.count()" + itemOrgStatusoSet.count());
    if (null != itemOrgStatusoSet) {
        itemOrgStatusoSet.deleteAll(11L);
        itemOrgStatusoSet.save();
    }
    itemOrgStatusoSet.close();

    MboSetRemote itemStatusoSet = server.getMboSet("ITEMSTATUS", server.getSystemUserInfo());
    itemStatusoSet.setWhere("itemNum='" + itemNum + "'");
    itemStatusoSet.reset();
    System.out.println("itemStatusoSet.count()" + itemStatusoSet.count());
    if (null != itemStatusoSet) {
        itemStatusoSet.deleteAll(11L);
        itemStatusoSet.save();
    }
    itemStatusoSet.close();

    super.delete(access);

    UdAtgUtil.print("Item delte success: " + itemNum);
}
